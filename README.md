# 📝 Todo CLI App

 A command-line Todo application built with Python.

## 🚀 Features

- Add, edit, delete tasks
- Mark tasks as complete
- Search tasks
- Task statistics
- JSON storage
- Input validation
- Colored terminal output

## 🛠  Tech Used
- Python 3
- JSON
- CLI (Terminal)

## ▶️  How to Run

````bash
python3 main.py
